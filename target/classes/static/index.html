angular made of 3 parts:

  • controller - the controller provides data to the page
  • service - the service adjoins to controllers, utilizing angular's dependency injection to chain together functions for use in the page.
  • directive - the directive attaches a behavior to an element

$scope is a special object, looked for with reflection, and assigned to each controller (service's and directive's can look at $scope too but it is not part of the page to be controlled, and so they don't have a special scope for themselves).
